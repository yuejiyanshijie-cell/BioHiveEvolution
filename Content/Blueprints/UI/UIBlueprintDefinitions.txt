// Widget Blueprint: WBP_UnitCard
// Parent Class: UUserWidget
// Description: 单位选择卡片，显示在底部中央

[Widget Tree]
Border (Root, with selection highlight)
├── UnitPortrait_Image
├── HealthBar_ProgressBar
├── UnitTypeIcon_Image
└── SetBonusIndicator (visible if has set bonus)

[Properties]
- UnitReference: ABioUnit*
- bIsSelected: bool

[Events]
- OnClicked → Select this unit
- OnDoubleClicked → Center camera on unit + select all of same type

// Widget Blueprint: WBP_BuildButton
// Parent Class: UUserWidget
// Description: 建造按钮，显示在建造菜单中

[Widget Tree]
Button (Root)
├── Background_Image
├── BuildingIcon_Image
├── BuildingName_Text
├── CostText (e.g., "50 Biomass")
├── HotkeyHint_Text (e.g., "[Q]")
└── ProgressBar (construction progress, visible during build)

[Properties]
- BuildingClass: TSubclassOf<ABioBuilding>
- UnitClass: TSubclassOf<ABioUnit>
- BuildCost: TArray<FBioResourceCost>
- BuildTime: float
- Hotkey: FKey

[States]
- Normal: Full opacity, clickable
- InsufficientResources: Grayed out, disabled
- Building: Progress bar animating
- Cooldown: Timer display

// Widget Blueprint: WBP_Minimap
// Parent Class: UUserWidget
// Description: 小地图，显示战场概览

[Widget Tree]
CanvasPanel (Root, 150x150px)
├── Background_Image (terrain overview)
├── FriendlyUnits_Canvas (green dots)
├── EnemyUnits_Canvas (red dots)
├── ResourceNodes_Canvas (yellow dots)
├── Buildings_Canvas (blue squares)
└── CameraViewport_Rectangle (white outline)

[Update Rate]
- Units/Enemies: Every 0.1 seconds
- Resources: Every 1.0 second
- Camera viewport: Every frame

[Interactions]
- Click → Move camera to clicked location
- Drag → Pan camera
