# BioHive Evolution v3 — 生物母巢进化

## 玩法简介

一款受「斗罗大陆」「46亿重奏」启发的 **3D 生物母巢进化生存游戏**。你在一个 500×500 的开阔草原上指挥一支生物军团，守护中央的水晶堡垒，抵御从四方杀来的敌军军团。

---

## 双模式操作

### 👁️ 观察模式（默认）
自由飞行浏览战场，无法操作单位。

| 按键 | 功能 |
|------|------|
| 鼠标移动 | 旋转视角 |
| WASD | 水平飞行 |
| Q / E | 升高 / 降低 |
| Q 键 | 切换到指挥模式 |

### 🎮 指挥模式（按 Q 进入）
战略俯瞰视角，可部署单位、管理资源。

| 操作 | 功能 |
|------|------|
| 左上角按钮 | 选择要放置的兵种（金色=已选中） |
| 点击空地 | 放置 1 个兵种 |
| 点击已有单位 | 进化该单位（需击杀达标+生物质） |
| Shift + 拖拽 | 框选区域，批量放置 4×3 = 12 个 |
| WASD | 平移地图 |
| 滚轮 | 缩放视野 |
| Q 键 | 切回观察模式 |

## 兵种系统（46亿重奏进化树）

| 类型 | Tier 1 | Tier 2（需5杀） | Tier 3（需15杀） |
|------|--------|-----------------|------------------|
| **drone** | 工蜂 — 采集资源 | 采集者 — 疾速采集 | 收割者 — 范围吸收 |
| **warrior** | 战士 — 近战肉搏 | 狂战士 — 嗜血 | 主宰 — 裂地斩 |
| **hydralisk** | 远程 — 远程输出 | 长枪 — 毒液弹 | 毁灭者 — 灭世射线 |

**进化方式：** 上帝视角下点击已有的己方单位 → 自动进化（消耗生物质）。

**进化效果：** HP ↑ 60%、攻击 ↑ 70%、体型增大、光环特效、解锁专属技能。

---

## 敌军系统

- 每 **20 秒**一波，每波 **5 个军团**
- 每军团 **25 人**：1 个 BOSS + 5 坦克 + 15 步兵 + 4 快速兵
- 从地图四边列阵行军，直扑水晶
- 后期波次：HP +35%/波，攻击 +18%/波，体型 +8%/波

---

## 资源与胜负

| 资源 | 说明 |
|------|------|
| 生物质 | 消耗资源，通过采集食物和击杀敌人获得 |
| 水晶 HP | 2000，降至 0 游戏结束 |
| 安全期 | 开局 30 秒无敌，屏幕倒计时 |

## 运行方式

双击 `biohive-evolution-v3.html`，任意现代浏览器（Chrome/Edge/Firefox/Safari）即开即玩。需联网加载 Three.js（首次缓存后离线可用）。
